

# Smart Contract Reference

Below is a list of the current live smart contracts across all of the networks that POAP supports.

For additional information about setting the minting network while creating a new POAP drop through the API, refer to the [minting-config endpoint](https://documentation.poap.tech/reference/patchdropsminting-config). Please note that you cannot set the minting network to Mainnet through this endpoint.
